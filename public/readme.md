# Student Loan Calculator UK

**The UK's Complete Student Loan Repayment and Strategy Platform**

Student Loan Calculator UK is the most comprehensive student loan repayment platform for UK borrowers. It covers every loan plan, regional system, employment scenario, salary structure, and lifecycle event — with calculators, guides, comparisons, region-specific support, and strategy frameworks.

---

## Overview

This site is built to help UK borrowers understand, predict, and optimize their student loan repayments across:

- **All UK loan plans:** Plan 1, Plan 2, Plan 4, Plan 5, Postgraduate
- **All UK regions:** England, Scotland, Wales, Northern Ireland, Crown Dependencies
- **All employment types:** PAYE, Self-employed, CIS, Umbrella/IR35, multiple jobs, bonus-heavy roles
- **All lifecycle events:** graduation timing, parental leave, career breaks, moving abroad, retirement
- **All repayment edge cases:** combined UG/PG loans, under/over deductions, P60/P45 reconciliation

## Site Philosophy

This platform is designed around:

- **Accuracy first:** Every calculator and guide is based on official SLC, HMRC, and devolved administration rules.
- **Scenario completeness:** Tools cover real-world borrower edge cases.
- **Strategy neutrality:** Guides explain outcomes without pushing a specific action.
- **Regional correctness:** Scotland/Wales/NI rules are treated as fully distinct systems.
- **Lifecycle awareness:** Borrowers’ decisions change over time; tools reflect that.
